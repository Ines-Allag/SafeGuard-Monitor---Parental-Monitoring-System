# 🔒 SafeGuard Monitor - Parental Monitoring System

**A responsible, modular, and cross-platform solution to help parents protect their young children online.**

SafeGuard Monitor allows parents to monitor activity on devices they own (phones, tablets, or computers) by logging keystrokes, tracking active windows, capturing periodic screenshots, and intelligently detecting sensitive or potentially risky content.

---

## ✨ Why SafeGuard Monitor?

What began as an academic project has grown into a complete parental monitoring framework. Inspired by real-world needs, SafeGuard Monitor helps parents ensure their children's digital safety while giving them supervised access to technology.

### Key Features

- **Keystroke Logging** with optional timestamps and active window titles
- **Periodic Screenshot Capture** of the active window (customizable interval)
- **Intelligent Content Detection** – automatically identifies emails, passwords, and suspicious keywords
- **Observer Design Pattern** for real-time alerts on sensitive data
- **Remote Monitoring** via gRPC (send logs and screenshots to a parent dashboard/server)
- **Cross-Platform Support** (Windows, Linux, and macOS)
- **Easy Executable Generation** – create standalone apps for deployment
- **Dual Interface**: Clean CLI + Modern Graphical User Interface (GUI)
- **Log Analysis & Export** – detailed classification with JSON output
